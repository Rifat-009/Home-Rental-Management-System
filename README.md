🏠 Home Rental Management System
A simple desktop application developed to make the home rental process easier for both property owners and tenants. The system brings landlords, tenants, managers, and administrators together on a single platform, making it easier to manage rental properties, bookings, payments, and maintenance requests.

This project was created as part of the Object-Oriented Programming II (C#) course at American International University–Bangladesh (AIUB).

📖 About the Project
Finding and managing rental properties often involves paperwork, phone calls, and manual record keeping. This project was developed to simplify that process by providing a centralized system where users can browse available properties, submit rental requests, manage bookings, and keep track of important information.

The goal of the project is to reduce manual work while making rental management more organized and user-friendly.
